==============================================================
 HOW EACH SERVICE IS INDEPENDENT / DEPENDENT
==============================================================

  auth-service:
    → INDEPENDENT. Owns auth_db. No calls to other services.

  product-service:
    → INDEPENDENT. Owns product_db. Referenced by other services
      but makes no outbound service calls.

  cart-service:
    → INDEPENDENT. Uses Redis only. No MySQL, no inter-service calls.

  order-service:
    → DEPENDS ON product-service (Feign client) to validate items.
    → Owns order_db.

  recommendation-service:
    → DEPENDS ON MySQL (recommendation_db) to track product stats.
    → No direct runtime dependency on other running services.

  frontend:
    → DEPENDS ON all backend services at runtime via REST API.
